# Mes

#### 项目介绍
MES2.0改版，针对界面UI、源码整理、功能升级。

#### 项目结构
* Mes.Client
  - Mes.Client.Model `实体模型、参数`
  - Mes.Client.Service `BE服务引用`
  - Mes.Client.Utility `常用操作类、枚举、基础类`
  - Mes.Client.Web `界面`
  - Mes.Client.UnitTest `单元测试`
* Mes.Server
  - Mes.BE `服务宿主`
  - Mes.BE.Objects `数据操作`
  - Mes.BE.Services `服务对外接口`
